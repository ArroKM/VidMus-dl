<h3>Installing</h3>

```java
$ apt update && apt upgrade
$ termux-setup-storage;mkdir $HOME/storage/shared/Vidmus
$ apt install git
$ apt install python
$ pip install requests
$ pip install tqdm
$ git clone https://github.com/ArroKM/VidMus-dl
$ cd VidMus-dl
$ python3 vidmus.py
```
